## Merge Files
Merge `aaa.txt` and `bbb.txt` into `ccc.txt`

## Run
```
$ make
$ make run
```

## Result
```
> cat aaa.txt        
hello world
this content
is from
file aaa.txt, wow

> cat bbb.txt
wow cpp programming wow
hello from bbb.txt yaaa

> ./main && cat ccc.txt    
hello world
this content
is from
file aaa.txt, wow
wow cpp programming wow
hello from bbb.txt yaaa
```
